# project-01-transaction-fraud-detection
💳 Banking Insights Assistant
A lightweight, modular GenAI‑powered banking analytics application built with Streamlit, Hugging Face models, Pandas, and a FAISS‑based RAG engine. This project demonstrates how modern AI can enhance financial analysis by combining traditional data processing with natural‑language intelligence.

Users can upload transaction data, explore automated spending insights, detect potential fraud signals, and ask natural‑language questions about their financial activity. The system retrieves relevant transactions using embeddings and RAG, then generates grounded answers using a Hugging Face LLM.

The codebase is fully modular, separating concerns into components for file handling, analytics, prompt templates, LLM interaction, retrieval, and UI rendering—making it easy to extend, maintain, and adapt for real banking use cases.
