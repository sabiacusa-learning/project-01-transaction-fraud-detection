# project-01-transaction-fraud-detection
💳 **Banking Insights Assistant**  
A lightweight, modular GenAI‑powered banking analytics application built with Streamlit, Hugging Face models, Pandas, and a FAISS‑based RAG engine. This project demonstrates how modern AI can enhance financial analysis by combining traditional data processing with natural‑language intelligence.

Users can upload transaction data, explore automated spending insights, detect potential fraud signals, and ask natural‑language questions about their financial activity. The system retrieves relevant transactions using embeddings and RAG, then generates grounded answers using a Hugging Face LLM.

The codebase is fully modular, separating concerns into components for file handling, analytics, prompt templates, LLM interaction, retrieval, and UI rendering—making it easy to extend, maintain, and adapt for real banking use cases.

## 🚀 Setup Instructions

### **1. Clone the Repository**
```bash
git clone https://github.com/sabiacusa-learning/project-01-transaction-fraud-detection.git
cd project-01-transaction-fraud-detection
```

### **2. Create a Virtual Environment**
```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### **3. Install Dependencies**
```bash
pip install -r requirements.txt
```

If `requirements.txt` is missing:
```bash
pip install streamlit huggingface_hub python-dotenv pandas
```

### **4. Create a `.env` File**
Create a file named `.env` in the project root:

```env
HF_TOKEN=hf_your_huggingface_token_here
MODEL_ID=meta-llama/Meta-Llama-3-70B-Instruct
```

### **5. Get a Hugging Face Token**
1. Visit [https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)  
2. Click **New token**  
3. Select **Read** permission  
4. Copy the token  
5. Paste it into `.env`

### **6. Verify Project Structure**
```
project-01-transaction-fraud-detection/
├── app.py
├── requirements.txt
├── .env
├── components/
│   └── genai_client.py
└── data/
```

### **7. Test LLM Connectivity (Optional)**
```bash
python test_llm.py
```

Expected output:
```
Fraud Risk: HIGH
Reason: Large transaction amount, unusual timing, and unfamiliar location.
```

### **8. Run the Streamlit Application**
```bash
streamlit run app.py
```

### **9. Open in Browser**
If Streamlit doesn’t open automatically:

```
http://localhost:8501
```

### **10. Analyze Transactions**
- Enter transaction details manually **or** upload a dataset  
- Click **Analyze**  
- Review the fraud risk and explanation generated by the LLM

## 🛠 Troubleshooting

### **401 Unauthorized**
- Ensure `HF_TOKEN` is valid  
- Confirm token starts with `hf_`  
- Restart your terminal after editing `.env`

### **Model Not Supported**
Use this in `components/genai_client.py`:

```python
InferenceClient(api_key=token)
client.chat_completion(...)
```

Do **not** use `provider="novita"`.

### **Token Not Loaded**
```python
import os
print(os.getenv("HF_TOKEN"))
```

If it prints `None`, `.env` is not being loaded.
